# Privacy

Internet Detective is designed as a local-first browser extension.

- No tracking.
- No analytics.
- No cloud storage.
- No hosted inference dependency.
- No OpenAI or Anthropic API calls.
- Analysis data is stored locally in the browser through IndexedDB.
- Users can delete local analysis data from the options page.

The extension analyzes visible page content on supported pages. It does not collect credentials, cookies, private messages, or hidden page data.
