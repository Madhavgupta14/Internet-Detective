# Spectra

Spectra is a Chrome extension that transforms public LinkedIn profiles into structured intelligence and ready-to-send outreach. Deterministic scoring runs locally in the browser; written insight and outreach are generated by an LLM through either a hosted cloud endpoint (default, no setup) or a local Ollama install (fully private).

## Features

- Chrome Extension Manifest V3
- React, TypeScript, Tailwind CSS
- LinkedIn profile extraction from visible page content
- Deterministic signal extraction and scoring (runs locally, always)
- Two LLM backends:
  - **Hosted cloud** (default): Llama 3.3 70B via a Groq proxy on Vercel — no user setup
  - **Local Ollama**: fully private, runs on the user's machine
- Resume-aware outreach generation
- Email finder (pattern-based, optional Hunter.io verification)
- Local IndexedDB persistence
- No analytics or tracking

## Architecture

```
Extension (content + background + UI)
  → Deterministic scoring (local, in the extension)
  → LLM insight/outreach:
       hosted  → https://internet-detective-proxy.vercel.app/api/analyze → Groq (Llama 3.3 70B)
       ollama  → http://localhost:11434/api/generate → local model
```

The hosted proxy lives in `server/` and holds the `GROQ_API_KEY` server-side, so no secret ever ships inside the extension.

## Local Development

Install dependencies:

```bash
npm install
```

Build the extension:

```bash
npm run build
```

Package the extension for distribution:

```bash
npm run package
```

The packaged ZIP is written to:

```text
release/spectra-extension.zip
```

Load the built extension in Chrome:

1. Open `chrome://extensions`.
2. Enable Developer Mode.
3. Click Load unpacked.
4. Select the `dist` folder.

## Hosted backend (default)

The extension ships pointing at the hosted Groq proxy, so users get high-quality analysis with zero setup. The proxy is a Vercel serverless function in `server/`:

```bash
cd server
npx vercel deploy --prod
npx vercel env add GROQ_API_KEY production   # paste your Groq key
```

Get a free Groq API key at https://console.groq.com/keys.

## Local Ollama (optional, fully private)

Switch the backend to "Local Ollama" in Settings. Default endpoint:

```text
http://localhost:11434
```

Default local model:

```text
qwen3:4b
```

Pull the model:

```bash
ollama pull qwen3:4b
```

### Allow the Chrome extension origin

If Ollama returns `HTTP 403` from the extension, Ollama is blocking the Chrome extension origin. Configure `OLLAMA_ORIGINS` with the installed extension ID, then restart Ollama:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File scripts/configure-ollama-origins.ps1 -ExtensionId YOUR_EXTENSION_ID
```

Without any LLM backend, Spectra falls back to deterministic scoring and a local outreach template.
